Remove site access from Internet Explorer users and offer them a link to download Chrome.

== Description ==
Internet Explorer has long been the bane of web developers and users alike for far too long. It's high time to kill off the defective software once and for all!

Take responsibility into your own hands by installing this plugin and restricting Internet Explorer from displaying your site's content. When a web surfer hits any page on your WordPress site with Internet Explorer, they are redirected to an appropriately colored static page with an error message urging them to install a more modern web browser. A link to Chrome is provided.

* Simple installation
* Effective results
* Brighten a programmer's day

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Re-save your Permalinks
